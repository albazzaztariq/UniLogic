Getting Started

From zero to running code in 5 minutes.

1Install Python 3.10+

The UL compiler is written in Python. You need Python 3.10 or later installed.

python --version

If you see Python 3.10 or higher, you're set. If not, download from python.org/downloads. On Windows, check "Add Python to PATH" during installation.

2Get the Code

git clone https://github.com/albazzaztariq/UniLogic.git
cd UniLogic

The compiler, REPL, and all tools are in this repository.

3Write a Program

Create a file called hello.ul:

function main() returns int
  print "hello world"
  return 0
end function

4Compile It

Compile to C (the default target):

# Compile and run in one step
ul run hello.ul

# Or compile to a specific target
ul hello.ul -t c        # produces hello.c
ul hello.ul -t python   # produces hello.py
ul hello.ul -t js       # produces hello.js
ul hello.ul -t llvm     # produces hello.ll
Wrappers included The repo includes ul.bat (Command Prompt) and ul.ps1 (PowerShell) wrappers that auto-detect your Python installation. On Linux/macOS, use ul.sh. You can also call python Main.py directly.

To compile C output to a native binary:

gcc hello.c -o hello

5Run It

# Native binary (after gcc)
./hello

# Python target
python hello.py

# JavaScript target
node hello.js

All three produce: hello world

6VS Code Extension

The extension provides syntax highlighting, diagnostics, hover definitions, autocomplete, and signature help.

# Build the extension package
cd LSP/vscode-ul
npm install
npx @vscode/vsce package --allow-missing-repository

# Install it
code --install-extension unilogic-0.1.0.vsix

Open any .ul file and the extension activates automatically.

7Try the REPL

The interactive interpreter lets you test expressions without creating a file:

# Terminal REPL
python REPL/repl.py

# Or try the browser REPL — no install needed
