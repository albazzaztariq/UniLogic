Features & Functionality

UniLogic's core features span memory safety, runtime configuration, compilation, and deployment. Together they form the full system that lets you write once and target any platform, any memory model, and any safety level.

Memory Safety

Mathematically-proven memory safety via MemProof. Formal verification with SMT solvers, digital signing, and functional correctness contracts. No lifetime annotations, no runtime overhead.

Dynamic Runtime

Per-file runtime configuration. Memory model, safety level, type system, integer width, and concurrency — declared at the top of each file, enforced by the compiler. Conflicts between files detected at compile time.

Translation / XPile

The compilation engine. Converts UL source to C, Python, JavaScript, or LLVM IR. Round-trip capable — style profiles control how target code maps back to UL syntax.

Deployment Targets

What platforms and languages the compiler can emit to. Native binaries via C/LLVM, portable scripts via Python, browser apps via JavaScript.
