UniLogic Glossary

Compiler

ul file.ulcompile file.ul to binary
ul file.ul --runcompile and immediately run
ul file.ul --emit-calso save generated C file
ul file.ul --emit-pyalso save generated Python file
ul file.ul --emit-jsalso save generated JavaScript file
ul file.ul --emit-javaalso save generated Java file
ul file.ul --astprint AST (debug)
ul file.ul --tokensprint token stream (debug)
ul file.ul --debugemit debug symbols (DWARF/PDB) into the compiled binary — stack traces include file names and line numbers; use --debug-stripped for production builds with no debug info

Transpiler — to UniLogic

xpile input.ctranspile C source to UL (prints to stdout)
xpile input.pytranspile Python source to UL
xpile input.jstranspile JavaScript source to UL
xpile input.javatranspile Java source to UL
xpile input.c -o out.ulwrite output to file (works with any source language)
xpile --c input.cexplicit C flag (inferred from .c by default)
xpile --py input.pyexplicit Python flag
xpile --js input.jsexplicit JavaScript flag
xpile --java input.javaexplicit Java flag

Variables

Variant

variant TypeNameEquivalent of C's union type

Pointers

int <p> = address(x)p is a pointer to x
Node <n> = address(obj)n is a pointer to obj
int <<pp>> = address(<p>)pointer to pointer
<p>the pointer value (used in expressions)
address(x)address of a plain variable
deref(<p>)value at the address p holds
p[0]same as deref — access value through pointer
<p> + 5pointer arithmetic — p advanced by 5 elements

Memory

size(Type)size in bytes of Type (like sizeof in C)
size(myVar)size in bytes of variable's type
change(expr)->Typecast expr to Type
change(expr)-><Type>cast expr to pointer-to-Type
malloc(size(Node))allocate heap memory for one Node
free(p)release heap memory at p
realloc(p, newsize)resize heap allocation at p
mem(<0xDEAD0000>)read/write at a raw address

Functions

function foo(int x) returns intfunction declaration
function foo(Node <n>) returns nonefunction taking a pointer param
function main() returns intentry point
return xreturn value
