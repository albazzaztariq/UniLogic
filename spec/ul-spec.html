UniLogic Language Specification

Version 1.1 — 2026-03-18

1. Lexical Structure

UTF-8 source. ASCII keywords and identifiers. Line comments: //. No block comments. Whitespace insignificant except in strings.

Keywords (Implemented)

Control: function end returns return if else while do for each in match default escape continue

Declarations: type object inherits new fixed const import from export

I/O: print prompt   Concurrency: parallel spawn wait lock unlock

Hints: nocache yield yields inline pack   Literals: true false empty

Logical: and or not   Memory: address deref left right memmove memcopy memset memtake memgive

Built-ins: size cast absval exit typeof

Types: int integer float double string bool none complex int8–int64 uint8–uint64 array list map arena file ok error some

Reserved (not implemented): iterate, teleport, portal, killswitch, equals, both1, both0, either1, delta, bitflip, negate, change, constant

User Types (Value Semantics)

type Point
  float x
  float y
end type

type Point3D inherits Point
  float z
end type

Objects (Reference Semantics)

object Circle inherits Shape
  float radius

  function area() returns float
    return 3.14159 * radius * radius
  end function
end object

Objects support methods, vtable dispatch, and expr.method(args) call syntax.

Result Types

function divide(int a, int b) returns int|error
  if b == 0
    return error "division by zero"
  end if
  return ok a / b
end function

3. Declarations

Function

function name(type param, type param = default, ...) [returns type]
  body
end function

Default parameters: type name = literal. Variadic: ... as last param. Multiple return: returns (type, type). Omitting returns = returns none.

Variable / Constant

int x = 5           // initialized
int y                 // zero-initialized
fixed int MAX = 100  // immutable (legacy)
const int MAX = 100  // immutable (preferred)

Object Declaration

object Name [inherits Parent]
  type field_name

  function method(type param) [returns type]
    body
  end function
end object

Methods called via instance.method(args). Compiler rewrites to pass instance as implicit first parameter.

Foreign / Local Import

import "stdio" function printf(string fmt, ...) returns int
import "utils.ul" function helper(int x) returns int

4. Expressions

Arithmetic: + - * / %. Integer division truncates toward zero.

Comparison: == != < > <= >=. Result is bool.

Logical: and or (short-circuit), not (unary).

Cast: cast(expr, type). Float-to-int truncates.

String interpolation: "hello {name}, age {age}". Expressions inside {} evaluated and converted to string.

Array comprehension: [x * x for x in range(1, 10)]

Pipe: data |> transform |> filter |> print. Rewrites a |> f to f(a).

Result propagation: expr? — if error, enclosing function returns it. If ok, unwraps.

Field/method: point.x, circle.area(). Index: arr[i]. Post: x++ x--.

Multiple return: (int a, int b) = swap(x, y)

10. Formal Grammar (EBNF)

program        = { dr_directive | annotation | declaration } ;

declaration    = function_decl | type_decl | object_decl
               | foreign_import | local_import | const_decl ;

dr_directive   = "@dr" IDENT "=" IDENT ;
annotation     = "@" IDENT [ "(" annotation_args ")" ] ;
const_decl     = "const" type IDENT "=" expr ;

function_decl  = "function" IDENT "(" [ param_list ] ")"
                 [ "returns" return_type ] body "end" "function" ;
return_type    = type | "(" type { "," type } ")" ;

type_decl      = "type" IDENT [ "inherits" IDENT ]
                 { field } "end" "type" ;

object_decl    = "object" IDENT [ "inherits" IDENT ]
                 { field | method_decl } "end" "object" ;
method_decl    = "function" IDENT "(" [ param_list ] ")"
                 [ "returns" return_type ] body "end" "function" ;

foreign_import = "import" STRING "function" IDENT
                 "(" [ param_list ] ")" [ "returns" type ] ;
local_import   = "import" STRING "function" IDENT
                 "(" [ param_list ] ")" [ "returns" type ] ;

param_list     = param { "," param } [ "," "..." ] ;
param          = type IDENT [ "=" literal ] ;
field          = type IDENT ;

type           = [ "<" ] type_name [ ">" ] | "array" type_name ;
type_name      = "int" | "int8" | ... | "uint64" | "float" | "double"
               | "string" | "bool" | "none" | "complex"
               | "list" | "map" | "arena" | "file" | IDENT ;

body           = { statement } ;

statement      = var_decl | const_decl | assignment
               | if_stmt | while_stmt | for_stmt | match_stmt
               | return_stmt | print_stmt
               | spawn_stmt | wait_stmt | lock_stmt
               | asm_block | "escape" | "continue" | expr_stmt ;

var_decl       = [ "fixed" | "const" ] type IDENT [ "=" expr ] ;
assignment     = expr ( "=" | "+=" | "-=" | "*=" | "/="
               | "%=" | "left=" | "right=" ) expr ;

if_stmt        = "if" expr body [ "else" body ] "end" "if" ;
while_stmt     = "while" expr body "end" "while" ;
for_stmt       = "for" "each" IDENT "in" expr body "end" "for" ;
match_stmt     = "match" expr { match_case }
               [ "default" body ] "end" "match" ;
match_case     = expr body ;

return_stmt    = "return" [ expr | "ok" expr | "error" expr
               | "(" expr { "," expr } ")" ] ;
print_stmt     = "print" expr ;
spawn_stmt     = "spawn" expr ;
wait_stmt      = "wait" ;
lock_stmt      = "lock" IDENT body "unlock" IDENT ;
asm_block      = "@asm" "(" IDENT ")" { ASM_LINE } "end" "asm" ;

expr           = pipe_expr ;
pipe_expr      = or_expr { "|>" or_expr } ;
or_expr        = and_expr { "or" and_expr } ;
and_expr       = not_expr { "and" not_expr } ;
not_expr       = "not" not_expr | comparison ;
comparison     = addition { ("==" | "!=" | "<" | ">"
               | "<=" | ">=") addition } ;
addition       = multiplication { ("+" | "-") multiplication } ;
multiplication = unary { ("*" | "/" | "%") unary } ;
unary          = ("-" | "address" | "deref") unary | postfix ;

postfix        = primary { call | index | field_access
               | method_call | "++" | "--" | "?" } ;
call           = "(" [ arg_list ] ")" ;
index          = "[" expr "]" ;
field_access   = "." IDENT ;
method_call    = "." IDENT "(" [ arg_list ] ")" ;
arg_list       = expr { "," expr } ;

primary        = INT_LITERAL | HEX_LITERAL | FLOAT_LITERAL
               | string_literal | "true" | "false" | "empty"
               | IDENT
               | "cast" "(" expr "," type ")"
               | "typeof" "(" expr ")"
               | "[" [ array_body ] "]"
               | "(" expr { "," expr } ")" ;

array_body     = expr { "," expr }
               | expr "for" IDENT "in" expr ;
string_literal = '"' { char | escape | "{" expr "}" } '"' ;
escape         = '\' ( 'n' | 't' | '"' | '\' ) ;

12. Semantics

Integer division truncates toward zero (C99). and/or short-circuit. Inner scopes shadow outer. Parameters by value; pointers for by-reference. ? in non-result functions prints error to stderr and exits with code 1. fixed/const = immutable. Falsy: empty, 0, 0.0, "", false. Method obj.m(args) rewrites to Type_m(obj, args). Pipe a |> f rewrites to f(a). Interpolation "x = {e}" rewrites to "x = " + cast(e, string).
