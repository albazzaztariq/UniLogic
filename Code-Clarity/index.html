Everything you need to start and keep learning

Every feature is designed around one goal: make programming understandable.

💬

AI Chat

Describe what you want in plain English. The model writes the code and inserts it into the editor. Bring your own API key — Anthropic, OpenAI, Google, or local models via Ollama.

✨

Clarity Change Log

Every AI-generated change produces a plain-English entry: what changed and why. Click any entry to highlight the relevant code and open a full explanation. No other tool does this.

💡

Highlight and Explain

Select any code in the editor. Right-click and choose Explain This. Get a plain-English paragraph explaining exactly what that code does, step by step.

🔧

Statement Expander

Nested and chained expressions expanded into clearly readable, one-operation-per-line code. Understand complex expressions without having to mentally parse them.

📊

Runtime Toolpane

A live panel showing the runtime model for your active language: memory management, threading, and compilation. For UniLogic, every property becomes a tunable dial.

📚

Language Guide and Tutorials

Progressive examples from Hello World to error handling. Some examples intentionally contain bugs for you to find. Active learning, not passive reading.

▶️

Build Panel

Run or build with one click. No terminal required. Results appear in the results pane. UniLogic projects support WASM and VM targets alongside native binaries.

🔍

Codebase Explainer

Two modes: Structure shows what files exist and how they relate. Execution traces what happens when the program runs — entry point, call flow, data paths.

How It Works

Three steps from zero to understanding.

1

Open

Start a new file. Pick your language. The editor is ready — no configuration, no terminal setup.

2

Build with AI

Describe what you want in the chat panel. The AI writes the code. You see exactly what was added and where.

3

Learn from Every Change

Click any Clarity entry to read what changed, why it works, and what concepts were used. Optionally try writing it yourself.
